Intro


What is a bookmarklet?


A bookmarklet is a bookmark stored in a web browser containing JavaScript commands which add new features to the browser.

To use one of the Google Search Console bookmarklets, follow the list below.



How to get started?


  1. Drag and drop one of the purple links below onto your bookmark bar

  2. Navigate to Google.de

  3. When on Google.de, click the bookmarklet on your bookmark bar

  4. This will open an extra tab

  5. Go back to the open Google.de tab and click the red alert in the URL address bar and click allow popups:



  6. You can now run the DE location bookmarklets from Google.de whenever you like.

Note: You can put the bookmarklets in a folder to prevent them from cluttering your bookmark bar.
